# Satellite Fetcher

## Setup
1. Instale Python 3.11

2. Crie um virtual environment
```console
$ pip install pipx

$ pipx install virtualenv

$ virtualenv venv
```

3. Instale dependências
```console
$ pip install -r requirements.txt
```

4. Crie arquivo `.env` e defina as variáveis de ambiente
```console
$ cp .env.example .env
```

5. Execute o servidor
```console
$ python server.py
```
